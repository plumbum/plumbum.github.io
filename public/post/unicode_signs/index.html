Подборка некоторых полезных символов из юникода.

Символ Название Код
₽ Рубль. Знак валюты U+20BD
✓ Check mark U+2713
✔ Heavy check mark U+2714
✅ White heavy check mark U+2705
⍻ Not check mark U+237B
☐ Ballot box U+2610
☑ Ballot box with check U+2611

Под X-сервером ввод символа с помощью его кода начинается комбинацией Ctrl+Shift+U.

Вставить символ в HTML документ можно через апмперсанд последовательность. Например © &#x00A9;
